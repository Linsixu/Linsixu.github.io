其中System.arraycopy(A,B,C,D,E)里面的参数的意思如下解释:
A: 要复制的数组A(源数组)
B: 数组A从B位置开始进行复制
C: 复制到数组C中(目的数组)
D: 数组C从D位置开始赋值(int C[D] = int A[B])
E: 复制的长度
